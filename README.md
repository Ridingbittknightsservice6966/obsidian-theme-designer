## 🎯 About this tool

The obsidian-theme-designer allows you to build custom visual themes for the Obsidian note-taking application. You do not need to write code to change your colors, fonts, or spacing. This tool provides a visual interface in your web browser where you adjust settings and see live previews of your changes. 

You can experiment with typography, font pairing, and color palettes to match your personal style. Once you finish your design, the tool exports the necessary files so you can apply them to your Obsidian vault.

## ⚙️ System Requirements

This application runs on Windows systems. Ensure your computer meets these basic criteria:

* Windows 10 or Windows 11.
* At least 4GB of RAM.
* A web browser like Chrome, Edge, or Firefox.
* An existing installation of Obsidian.

## 🚀 Creating your first theme

Launch the application using the shortcut on your desktop. The tool loads a workspace where you manage your visual styles. 

### Choose a base theme
Select a starting structure from the templates menu. These templates provide a solid foundation for your layout. You can adjust every element later.

### Adjust your colors
Use the color picker tools to modify the background, text, and accent colors. The preview window updates your changes as you move the sliders. 

### Edit typography
Select fonts from the list to test different pairings. The tool shows how your headings and body text look together. Use the spacing controls to adjust the gaps between lines and paragraphs.

### Manage your snippets
The tool organizes your design as a CSS snippet. Obsidian uses these snippets to apply your visual changes. You can enable or disable specific snippets to test different variations of your design.

## 📂 Applying your theme to Obsidian

After you finalize your design, you must move the file into your Obsidian vault folder.

1. Click the "Export" button in the application.
2. Save the resulting file to your computer.
3. Open Obsidian.
4. Go to Settings, then Appearance.
5. Click the folder icon next to "CSS snippets."
6. Drag your exported file into that folder.
7. Return to the Obsidian settings menu and toggle the switch for your new theme.

## 💡 Troubleshooting common issues

If you encounter problems during the process, check these common fixes:

* **App fails to open:** Restart your computer and run the installer again.
* **Theme does not appear in Obsidian:** Ensure you saved the file in the correct snippets folder. Obsidian requires the file to be placed specifically within the hidden `.obsidian/snippets` directory of your vault vault.
* **Colors look different:** Some Obsidian plugins override custom themes. Disable other appearance plugins if you notice conflicts with your design.
* **Font issues:** Ensure the fonts you choose are installed on your Windows system or are standard web fonts.
* **Updates:** Check the download page periodically for new versions that include better support for recent Obsidian updates.
